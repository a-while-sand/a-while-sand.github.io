Jmeter压力测试流程

1、先创建【测试计划】

2、右击【测试计划】创建【线程组】

3、右击【线程组】创建【http请求】

4、右击【http请求】创建【Graph Results】和【Summary Report】

5、增加默认参数：

【线程组】 -> 添加 -> 【配置元件 -> 选择 【HTTP请求默认值】

​ 设置请求url和端口之类

6、增加CSV数据文件设置（可设置多用户访问）

【线程组】 -> 添加 -> 【配置元件 -> 选择 【CSV数据文件设置】

选择【文件名】，找到指定的文件

设置【变量名称】：username,token
