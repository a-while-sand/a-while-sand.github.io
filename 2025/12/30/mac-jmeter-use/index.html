总体思路：

1、下载

2、安装

3、配置

4、查看报告

一、在官网下载Jmeter

1、https://jmeter.apache.org/download_jmeter.cgi，选择5.6.3版本，点击zip格式文件下载。

二、安装

1、将下载的zip文件解缩本机上/opt目录

cd /opt
双击 apache-jmeter-5.6.3.zip 自动解压

2、apache-jmeter-5.6.3，是要求java1.8以上

java -version
# 版本是大于8就行，11、17、21之类都行

3、运行

cd /opt/apache-jmeter-5.6.3/bin
./jmeter.sh

弹出jmeter窗口，就成功了。

三、配置

1、进入界面时，默认有【测试计划】

2、右击【测试计划】，选择【线程（用户）】，再选择【线程组】，配置100个线程数,Ramp-up是指多久启动100个线程

3、右击【线程组】创建【http请求】，填写好请求地址和参数

4、右击【http请求】创建【Graph Results】和【Summary Report】

5、增加默认参数（若需要登录就增加）

【线程组】 -> 添加 -> 【配置元件 -> 选择 【HTTP信息头管理器】

​ 新建Cookie的名称, 值将登录后Cookes值复制过来，就可以免登录测试

四、查看报告

点击运行按钮

javaMacJmeter
